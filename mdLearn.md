## 3. 代码块

代码块通过两行 ``` 符号框出，如果你写的代码是某种语言，那么可以在第一行末尾加上这个语言的名字，代码块内的代码就会执行对应的高亮语法，例如python

```
print("hello world")
```

```python
print("hello world")
```

## 5. 列表
有序列表，输入数字，加一个句点，然后加空格即可；可以缩进空置多级列表；
1. 123
2. 456
3. 789
   1. 123
   2. 456
   3. 789
      1. 123
      2. 456
      3. 789

无序列表，输入-，然后空格
- 123
  - 456
    - 789
